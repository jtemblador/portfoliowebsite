How should the explore/portfolio toggle work?

When the user clicks "Explore", portfolio content fades out and the full interactive viewer activates. How should the toggle look and behave?

A

Small icon button (corner)

A subtle telescope/compass icon in a corner. Unobtrusive, discoverable by curious visitors. Clicking it fades out portfolio content and reveals full viewer UI (search, time controls, overlays). A matching "Back" button returns to portfolio.

B

Text button in the sidebar

An "Explore the sky" link in your sidebar nav, alongside About/Projects/Experience. Feels intentional — like a portfolio feature you're showing off. Transitions the same way.

C

Floating pill button (bottom)

A small floating "Explore Sky ✦" pill centered near the bottom. Always visible, feels modern. In exploration mode it becomes "Back to Portfolio".
