# Jose T. Temblador — Portfolio

Personal portfolio website, live at [josetemblador.com](https://josetemblador.com).

Software developer studying Computer Science at CSUDH, graduating May 2026. Currently building OrbitWatch, a real-time satellite orbit tracker with a custom C++ SGP4 engine.

## Features

- **Live star viewer background** — Real-time night sky from Los Angeles rendered behind the portfolio. 15,598 stars, constellation lines, planets, Sun, Moon, and Milky Way. Click the compass icon to enter full interactive exploration mode with drag-to-pan, zoom, search, time controls, and object info popups.
- **Dark/light theme** with smooth transitions
- **Vanilla stack** — HTML, CSS, JavaScript. No frameworks, no build step.
